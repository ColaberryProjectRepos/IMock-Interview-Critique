Creating Calculated Columns 13

Calculated Columns:

1. Day_Month Column 

Right click on Date table - Select New column then type the DAX code and click on commit (Check mark) to create the column.

This is the DAX formula--- Day_Month = FORMAT('Date'[Date],"d mmmm")

Follow the same steps to create Rolling12Months calculated column.

2. Rolling12Months Column:

This is the DAX formula--- Rolling12Months = IF(DATE(YEAR(TODAY()),MONTH(TODAY()) -12, DAY(TODAY())) < 'Date'[Date], 1,0)


Difference between calculated column and calculated measure:
